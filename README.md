Учебный проект "Six cities"
==========
Сложный проект на ReactJS и TypeScript:
* верстка компонентов через JSX
* типизация
* unit-тестирование Jest
* React Hooks, React Router
* сборка через CRA

Деплой проекта: https://six-cities-training.netlify.app/
